Personal Expense Tracker

A beginner-friendly Python project to manage daily expenses using basic Python concepts such as variables, data types, lists, dictionaries, functions, loops, and modules.

Features
Add a new expense
View all expenses
Calculate total spending
View category-wise spending
Find the highest expense
Delete an expense
Menu-driven interface
