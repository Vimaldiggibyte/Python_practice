# Personal Expense Tracker

A simple Python project to manage daily expenses while practicing core Python concepts.

## Features

* Add an expense
* View all expenses
* Calculate total spending
* View category-wise spending
* Find the highest expense
* Delete an expense

## Project Structure

```text
src/
└── day_1/
    ├── main.py
    ├── welcome_start.py
    ├── input_file.py
    ├── data_handling.py
    ├── display.py
    └── calculation.py
```

## Modules

| File               | Description                   |
| ------------------ | ----------------------------- |
| `main.py`          | Starts the application        |
| `welcome_start.py` | Displays the menu             |
| `input_file.py`    | Reads user input              |
| `data_handling.py` | Manages expense data          |
| `display.py`       | Displays expenses             |
| `calculation.py`   | Performs expense calculations |

## Python Concepts Practiced

* Variables & Data Types
* User Input
* Lists & Dictionaries
* Conditional Statements
* Loops
* Functions
* Modules & Imports
